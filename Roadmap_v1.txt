Roadmap for the Management System project
Starting form the MainForm groups implementation

1) Implement the display of the groups in the MainForm

2) Implement file editors:
  2.1) Text editor
  2.2) Database table editor (from connected database and *OPTIONAL* locally stored tables)
  2.3) XML editor
  2.4) Image viewer

3) Implement the export system; formats:
  3.1) .txt
  3.2) .pdf
  3.3) .xml (if the input file can be converted into an XML file)
  3.4) .tbl (custom format for locally stored copies of databse tables)

4) Implement a form editor (*OPTIONAL*)

5) Implement a print preview and print function (*OPTIONAL*)
