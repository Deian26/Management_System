Locally stored Database Table file format : XML format (.tbl extension)

Format:
<table name="">
	<columns column1="" column2=""/>
	
	<rows>
		<row value1="" value2=""/>
	</rows>
</table>

Format example:
<table name="Table1">
	<columns column1="Column1" column2="Column2"/>
	
	<rows>
		<row value1="value1" value2="value2"/>
	</rows>
</table>
