Locally stored Database Table file format : XML format (.tbl extension)

Format:
<table name="">
  <column name="" type="">
    <row index= value= >
  </column>
</table>

Format example:
<table name="table1">
  <column name="column1" type="nvarchar(20)">
    <row index=1 value="value1">
    <row index=2 value="value2">
  </column>
</table>
